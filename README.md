# 雪窗文稿

## 废园集

### 长岛城记

长岛城总与别处不同的，第一是凌空的轻轨，从地下钻出，运行在云霄。连绵的粗钢柱将长条的轨道托起，一似“水面清圆，一一风荷举”。至Queensboro站时有极大的由南向东的拐弯，悬空的轨道边的护栏低矮，像是百草园“光滑的石井栏”般。又由于要提供向心力的缘故，车厢需要向东南倾侧，车窗外便是几十米之下的道路和商肆，使乘客感到如履如临的惊险。当列车缓缓驶入Queensboro站时，西边是横跨罗斯福岛的大桥，古朴而密集的钢筋拉索充满第二次工业革命的气息；更远处是曼哈顿的高楼华厦，一如大学生站在拉雪兹神父公墓的高处眺望，而且立志要征服的一样。总之梦幻般的，林林总总的工业感的震憾。帝国壮年的遗物，最宜在夕阳下瞻仰；西风残照，汉家陵阙，“遂关千古登临之口”。杜诗说“闻道长安似弈棋，百年世事不胜悲”。曼哈顿公然又是长安的棋盘了。王侯第宅、文武衣冠泰半依然歆享着当年的荣光。我缓缓走下Queensboro车站，到地面上，藏身在人流中。

2018年8月4日。

### 哥伦比亚大学记

因故友的缘故，哥伦比亚大学也曾是我梦想的大学，又岂意抵美五年后方初访此地。她的形容是曾见过的，但不似贾宝玉做神瑛时所见，不过从在线资讯中所见罢了。她在高楼间架的纽约地图上止据尺寸，主要的楼也是宣传画册上所习见。由是方在地铁上时，早已有了印象。从一号线升阶至路面，不料已是哥大的西门。两边分列着西方塑像上的健壮的高加索男子。其居北者手捧写着SCIENTIA的板。居南者我未细观。自东门入数十步地，北侧即是向时所见的楼，希腊神庙式的facade。我读其上的刻字第一次知是哥大的图书馆。像哈佛和耶鲁一样，门前坐着位老爷。南面是Butler图书馆，上刻列位罗马先哲的名字。这便是美国的第五所大学。图书馆上的铭文既写着国王学院，公众利益，也写着全能的上帝。与耶鲁不同的，旗杆上悬挂着蓝旗，而非美国国旗；铜的基座是相同的，覆满了铜绿，究其纪年，是十九世纪近末。一切印证她是希腊罗马的一脉存续——这是如今世界文明中的“显学”。民族非唯仰赖祖先而自立，更仰赖子孙以图存。如今世界上少数民族的文学，必有不减古今文豪高处的，然而比起中国人的熟知英法俄各国的作者，只能说是泯灭无闻了。又何知今日写下的絮叨的梦呓他日不一如西夏、契丹的密语，褒禅山的石碑？遐远的忧心，又岂今所独擅，自永嘉、天宝、靖康、德祐、崇祯、道、咸以降，未尝不时时深击中国的识字者。罗马思想家接受万众的膜拜和香烟，坐享当时未及见的的子孙，仗无远弗届的航海，添上的非唯冷猪肉的殊荣。纽约夏日静谧的阳光照在他们名字前的方正的草坪上。

2018年8月5日。

### 纽约地铁记

#### 一

荒芜的人海，总似车辚马萧里的乱离。所谓荒芜，无非觉得连草菅也不见，唯有古老腌臜的墙上匆匆的面容的剪影。长吉的诗“夜闻马嘶晓无迹”。

#### 二

路灯下。迎面女青年刚买菜回来，身上还残存着途中的车灯和月光，我由是知道是哪家超市，只有那条路有独特的灯光和月光。但我显然又在做着夜幕下的幻想——这不过是第二次来到长岛城。邻家的青年穿着背心和尼龙球裤，在高楼上的玻璃外墙里挥汗如雨地篮球。我仰望见橙黄的灯光朗照的木地板，以及他冲向的篮架，伸上篮筐把球拨入。下面一层是女生在窗边看书，兰花静静坐在一旁，有腴红鲜丽的灯。人们忙于生活，洗衣服、看书、打篮球、桌游，在一层层、一幢幢高楼里。我匆匆赶往地铁站的路上。

#### 三

此时我已坐下在地铁里，回想刚刚看到夜色里的多幕剧。所见都是中国的面孔，大抵是中国青年的社区，在长岛城。Queensboro附近的铁路在高空蜿蜒着，以是他们便漂洋过海在厨房和卧室里望见火车在Queensboro的天际穿行。我想见前哲所梦的枣红色的梦，像茜雪般飘在纽约和长岛城的高楼华厦间。我甚至想见耶诞节、感恩节围坐炉火的温馨了，像在冬天的黑幕里秉着烛光——北极的驯鹿将要来临了，还有伴着极光的铃声。火锅的氤氲的水汽几乎把我的镜片模糊。隆隆的地铁声。我还在独自用手机写点文字。World Trade Center站远远未到。

2018年8月11日。

### 肃威路

二十年前，我尚住在福州裴仙宫（老福州人呼之“仙宫”）旁的一座新村的顶层。那时候肃威路上的木棉树还有六七层楼高，二三人合抱的树围，至春日枝头挤满饱满厚实的朱红的木棉花，至暮春则化为满天飞舞的棉絮，几若雪景。夏日，台风（福州人称为“风台”）时时来，经年间吹折了木棉树的壮枝，也有为园林部门所锯去的，因而后来总显得稀稀落落的。我也不知在去国暌违的六年间，它们究竟如何了。想见同时是福建省道教协会的仙宫当是尚在的，它的大门西向，临肃威路，背后有一株千年的榕树，树冠覆盖了周围许多楼宇的屋顶。仙宫前南侧是国旗台，北侧是戏台。所谓戏台者，上有顶棚，下是灰水泥垫高的一尺的平台。往昔耆老尚众时，时时有戏班至，轮番地唱戏。戏台上用屏隔断前后台，两侧垂下投影的唱词，是毛笔的楷字。看戏需自备椅子，有时戏连唱几天，住在附近的可每日带椅子，也有将椅子锁在一起的。戏是夜唱，日中经过，极盛时能见到十几排椅子。无有戏班时，也或有讲评话的，通常不到十人，说者和听者都坐在戏台上。现已不再有戏台了。只是旁边还有经岁缭绕的香烟，将壁熏出褐黄色。畴日福州的唱戏也是有盛景的，恐今人多不知悉了。譬如说元帅庙的虎婆宫，以及八年前我于于山天君殿（于山是一座小丘陵，就在福州市中心的五一广场北面，往日其南山麓是南城墙），在微雨的山上所见白昼唱戏的，山岚、雨烟、焚香、戏唱交融于此。我知道这些，惟家中老人嗜戏故，常常在鼓楼区的各处看戏。后来她是鼓楼区最年长的老人，本来年年请护士来家中挂瓶，除去血栓。她仙逝后我才知她于几年前执意不再挂瓶了。这两三年间，她时时在电话里盼着我回来。四年前的仲秋，我随口说“今年冬天会回去”。她每日撕下日历的好几页，直见到冬天来临的那页。

肃威路向北即遇到东西向的鼓西路。它发自鼓楼以西，故得名。我记得二十五年前，也是我初记事时，在此丁字路口有一个摊位，每旦有老者设蜂窝煤炉，上面煮着袋装的牛奶并茶叶蛋。那是家中老人送我去幼儿园每日的途中。后来兹地立起了新的住宅楼。辽鹤令威，烂柯王质，莫不如是。

2020年12月6日。

### 读阮籍《咏怀诗》其六十五（“王子十五年”）

阮籍《咏怀诗》其六十五云：

王子十五年，游衍伊洛滨，朱颜茂春华，辩慧怀清真。焉见浮丘公，举手谢时人。轻荡易恍惚，飘飖弃其身。飞飞鸣且翔，挥翼且酸辛。

这段故事本自《列仙传》：

“王子乔者，周灵王太子晋也。好吹笙作凤凰鸣，游伊洛间，道士浮丘公接以上嵩高山上。三十年后，求之于山上，见桓良，曰：‘告我家，七月七日，待我于缑氏山头’。至时，果乘白鹤驻山头。望之不得见，举手谢时人，数日而去。”

十五年，即十五岁。“举手谢时人”者，告别当时的人。我以为若论五言诗的成就，阮籍是最优秀的作者之一；事实上，他的风格即代表了五言古诗的正统风格。阮籍的诗在象征和奇思妙想上，是很具有现代性的；若是写以白话文，亦是很妙的白话诗，却无法用字精炼如此。此诗描绘了王子乔告别时人的感伤场面。最后两联先是只用十字写王子乔飞升的情状，后居然是“飞飞鸣且翔，挥翼且酸辛”。按理说既是成仙，何必沉痛如此。此诗当或悼亡或另有所指，有前人指曹髦。黄侃评价此诗说：“神仙竟无可信，子晋缑岭之游，人传仙去；然飘摇恍忽，竟与死去何殊！观于此诗，而阮公忧生之情，大可见矣。”阮籍诗中的王子乔其实是死去了，不仅是死去了，而且是早夭了（“王子十五年”，诗中的故事里并没有“三十年后，求之于山上”云云）。“挥翼且酸辛”已道破天机，所谓升仙之事，无非是粉饰和文字的迷障罢了。百代之下，读者当能领悟最后十字中对人间的无限留恋。否则，成仙当是何其美好的事，阮籍于几百年后何必感慨。人间有多少这样身不由己的事情，倘若王子乔不遇浮丘公，又何必无限感伤地举手谢时人呢？

2021年1月1日。

## 宅兹集

### 小引

首先向对机器学习感兴趣而对旧诗不感兴趣的同学致歉。很抱歉干扰了时间线。以下是2019年到2020年我的五言古诗的删存，原題“己亥庚子岁五古删存”。在此一个有更广大受众的平台仍用干支，未免文人酸腐了些，所以从何尊的铭文里割裂出了二字，重题为“宅兹集”。几天前，有朋友问我，自我定位是“士农工商”中的哪一个。我不假思索回答说："是‘工农士商’的‘工’。"读工科是一件很可喜的事情。作为工程师，我也偶尔写一些不能算是诗的东西以自娱。曹丕《与吴质书》结尾说：“顷何以自娱？颇复有所述造不？”陶潜《五柳先生传》说：“常著文章自娱，颇示己志。”说的就是这个。于是我把经年累积的篇章稍加编次，就是下面的这些，聊供方家一哂。以上是小引。

2020年12月16日。

### 为北社社课作中秋诗二首限侵韵

#### 其一

自予之海国，竟成久滞淫。既知是中秋，无宴何所斟。街灯立多时，承露忽已深。幸有二三子，以骋少年心。兹地网球场，铁网树萧森。客子眇天末，盈月有微忱。海推苍冥大，厥光安可任。素心如堪表，照我千山岑。

#### 其二

煮羹室中暗，窗上水涔涔。还念广寒节，故加鱼肉燖。商风因隙入，凄序乃已深。人生容易老，怆然南陔心。陟屺生我迟，感此泪不禁。

### 忆昔

忆昔年少日，庭闱犹壮龄。市朝教识字，携我出东城。喧卑乃如此，颇识鸟兽名。日夕暗归履，长楸广且平。焉知二十年，母已不能行。家父闲养蚕，桑树五枯荣。春蚕握盈手，桑叶何青青。还似童稚时，拊我畜我情。吾生诚梦幻，岁月使人惊。

### 弄璋

弄璋与弄瓦，人皆谓之喜。托身此喧卑，谁问可怜子。竟世为劳薪，摧折终以死。伤彼往生人，百万化为鬼。万鬼如湍泄，死去知何似。但似未生时，冥蒙不辨耳。

### 避疫

避疫岑楼上，恍然七十日。日字一户窗，延春入虚室。窗狭春何广，庭圃草已密。四时成新岁，惟人独自失。客居罹多故，蛮酋颇无术。我每闻其过，则喜不为恤。还聆沧溟落，遥睎月将出。永怀东海滨，泰山崇且崒。

### 不眠

生民既荡析，今予尚流离。顷日多风雨，夏气日以衰。蓂落去何返，旧雨与之随。人生当如此，欲眠梦已希。永夜久冥冥，起视夜何其。迢迢独夜舟，晨舣会有时。

### 读今人诗

狼藉今人诗，辄言佛与酒。二者吾弗信，于我竟何有。矧不慕清季，杌陧嗟老丑。罔择我生年，造化已相厚。仰视日光辉，照我东南牖。壮齿鬒未变，周京亦昌阜。

### 学诗（其二）

风骚久末法，屈指三世许。骚人荣宿草，吾侪竟谁与。欣然立斯世，而观新物举。飒沓如繁花，采撷作五古。谁知裳作带，石顽难炼补。况今万象殊，陈陈安得所。殷忧故有待，竟日春雪阻。承平逾百年，天将生李杜。佗年吾未老，当见盛时语。念此忽掩卷，长喟泪如雨。

### 后浪诗

曩昔之有君，犹可抑阀阅。诸夏今也亡，寒素何堪说。渊明责厥子，忧心其惙惙。读书竟何益，鲍诗徒在舌。我祖本陈涉，怅恨耕亦辍。乃言鸿鹄志，横游于寥泬。何当起朔方，侯景酬壮节。王谢高非偶，澄江已骚屑。
